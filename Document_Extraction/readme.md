Document Extraction 


This project extracts the relevant fields from PAN card such as name,PAN number,DOB of person.


The following steps are used :

. Image reading
. Image binarization
. Image cropping
. Extraction of data from Image


The requirements of this project are :

. Python (3.6.9)
. PILLOW
. OPENCV
. Pytesseract
. ftfy


The example data for PAN are provided in data folder.


For running the script just open a terminal and type :
 
	python main.py


This project can be expanded to extract fields from AADHAR Card, Voter's ID, and other ID's
